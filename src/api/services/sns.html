Amazon SNS is one of most flexible services that integrates natively with AWS Lambda. It allows many subscribers to topics, at high volume, with filtering on message attributes, cross account and cross region integration is possible!

Key Info & Limits

  • 12,500,000 subscriptions per topic (soft limit)
  • 100,000 topics per account (soft limit)
  • 200 subscription filters (soft limit)
  • 30,000 publish transactions per second (soft limit)

Bonus: Amazon CloudWatch Events

Amazon CloudWatch Events is a dark horse in this race. While not typically mentioned beyond a source of configuration or monitoring events, you can pass custom events through the system. It offers tremendous capabilities with filtering, and can send to many different destinations, including directly to other AWS Accounts' CLoudWatch EventBus'.

Key Info & Limits

  • 400 PutEvents Requests per second soft limit (soft limit)
  • 10 entries per PutEvents, 256kb per request
  • 750 rule target invocations per second (soft limit)
