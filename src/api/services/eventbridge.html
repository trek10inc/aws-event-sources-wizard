Amazon EventBridge (previously CloudWatch Events) is the new kid on the block and is great for serving as the communication fabric of your organization. It offers tremendous capabilities with filtering, and can send to many different destinations, including directly to other AWS Accounts Event Buses.

You can learn more about why EventBridge is awesome from Trek10.

Key Info & Limits

  • 400 PutEvents Requests per second soft limit (soft limit)
  • 10 entries per PutEvents, 256kb per request
  • 750 rule target invocations per second (soft limit)
