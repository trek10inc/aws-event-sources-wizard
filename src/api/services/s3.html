Tried and true, using Amazon S3 to pass around objects of data is a great practice. Especially with the ability to directly trigger AWS Lambda functions for new or updated objects on the whole bucket, or particular object name matches.

Key Info & Limits

  • 1 byte to 5 TB per object
  • Event delivery in seconds, occasionally minutes
  • 100 subscriber limit
  • 3,500/sec PUTs per prefix
  • 5,500/sec GETs per prefix
