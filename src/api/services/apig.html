If you are building an API for others for frontend applications, or others (third parties) to consume, Amazon API Gateway is the defacto choice within the AWS ecosystem. These days API Gateway even offers real-time communication via WebSocket support!

API Gateway can meet a broad range of requirements across authorization, thottling & rate limit plans, mapping and translation of requests from legacy APIs to a modern destination and format and more.

Key Info & Limits

  • Soft limit of 10,000 requests / sec
  • 29 second response timeout (your lambda must respond within the time frame)
  • 300 routes / API
  • 10MB payload limit
